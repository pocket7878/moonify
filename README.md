# moonify

Convert image into moon 🌓 emoji

## Install 

```
go get -u github.com/pocket7878/moonify
```

## Usage

```
./moonify <image_file> <width> <height>
```

## Example

```
./moonify ./gopher_pilot.png 32 32
```
